Sobre este sitio

Este sitio se alimenta de datos de una -somekindof- estación meteorológica. Los datos se capturan cada X segundos y el sitio se actualiza cada X minutos. Para ello se utiliza el programa . La estación se compone de un anemómetro, un pluviómetro y un sensor termo-higrómetro colocados de la mejor forma para obtener datos fiables.
